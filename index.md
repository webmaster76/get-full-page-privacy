# Privacy Policy for Get Full Page

Last Updated: June 2026

## Overview

Get Full Page is a browser extension that allows users to capture full-page screenshots of websites they choose to visit.

We are committed to protecting user privacy.

## Information We Collect

Get Full Page does not collect, store, transmit, sell, or share personal information.

The extension does not collect names, email addresses, payment information, passwords, browsing history, or location data.

## Website Content

When a user initiates a screenshot capture, the extension temporarily accesses the content of the current webpage solely to generate the requested screenshot.

Processing occurs locally within the user's browser.

Website content is not transmitted to external servers.

## Storage

The extension may store user preferences locally using Chrome storage APIs.

These settings remain on the user's device.

## Data Sharing

Get Full Page does not sell, rent, transfer, or share user data with third parties.

## Remote Code

The extension does not load or execute remote code.
